World Happiness Report Streamlit App

Overview:
This is a Streamlit web application that visualizes and analyzes the World Happiness Report data from 2015 to 2019. The app allows users to explore happiness scores, rankings, and correlations across countries and years.

Included Files:

run_me_first.py
Prepares and cleans the dataset by combining yearly CSV files into a single, standardized file called combined_happiness_data.csv.

run_me_second.py
The main Streamlit app. Launch this to explore the data interactively.

data/
Folder containing the raw CSV files for the years 2015–2019.

combined_happiness_data.csv
The cleaned and merged dataset generated by run_me_first.py.

How to Run the App:

1. Make sure required libraries are installed:
	pip install pandas streamlit plotly

2. Run the data preparation script:
	python run_me_first.py

3. Launch the Streamlit app:
	streamlit run run_me_second.py

App Features:

Select a year to explore

Filter by region or country

View overall happiness rankings

Display top or bottom N countries

Visualize correlation between happiness score components
